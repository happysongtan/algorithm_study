## 풀이 문제
- 주소

## 풀이 아이디어
- 각 노드들의 연결 여부를 확인하는 arr, 해당 노드의 방문여부를 확인하는 visited를 boolean으로 선언해준다
- 연결 수를 link 노드 갯수를 count로 선언 받는다.
- 각 노드를 입력 받으면 해당 노드의 연결 여부를 arr로 파악한다.
- 방문여부를 기준으로 dfs를 실행
- dfs에서 해당 node에 배열을 탐색하여 방문여부를 파악하여 새로운 root 발견시 count를 증가시켜준다. 

## 시간 복잡도
-O(N^2)
 
## 느낀점
- 배열로 풀었으나 리스트로도 풀 수 있을 것 같음.
- 아직 dfs의 개념이 익숙하지 않아서 다시 풀어봐야할듯
