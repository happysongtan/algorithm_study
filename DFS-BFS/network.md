#풀이 문제
- [주소](https://school.programmers.co.kr/learn/courses/30/lessons/43162)

##풀이 아이디어
-방문 여부를 boolean 배열을 통해서 관리<br>
-미방문 경로를 발견시 새로운 경로로 판단 count를 +1해주고 verification함수를 실행<br>
-함수 내부에서는 기준점 index를 기준으로 연결점을 확인<br>
-연결점을 확인하였는데 해당 연결점이 미방문 지점이면 연결점을 기준으로 다시 함수 실행<br>

##시간 복잡도
-O(n^2) <br>
-n x n 배열을 기반으로 모든 연결을 확인함

##느낀점
-문제를 맞닿아 있는 배열만 연결 취급으로 착각하여 기본 구조의 방향성을 고쳤음.<br>
-기준점을 잡고 방문/미방문 지점의 판단을 boolean을 사용해서 관리하였음.<br>
-새로운 경로로 들어설때 즉 count가 늘어나는 지점을 연계 생각이 부족했음<br>
