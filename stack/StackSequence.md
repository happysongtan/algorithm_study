##풀이 문제
- [주소](https://www.acmicpc.net/problem/1874)

##풀이 아이디어
-입력받은 숫자가 나올때까지 stack에 숫자를 push하고, 해당 숫자가 나오면 pop 아니면 push<br>
-만약 입력 숫자보다 count가 더 높아지면 반복문을 멈추고 NO 출력<br>

##시간 복잡도
-O(N)

##느낀점
-불가능한 경우를 판단하는 if문의 위치를 선정하는데 어려움이 있었고, push연산이 한번더 실행되는 상황이었음.<br>
-위치를 pop다음 위치에 오면 적절하다고 판단하여, 위치를 변경 후 완료하였음.<br>
-NO출력을 index로 관리하였는데 index보다는 다른 변수명을 사용하는 것이 더 적절함<br>
