
##풀이 문제
- [주소](https://www.acmicpc.net/problem/10828)

##풀이 아이디어
-문제 명령에 맞게 if를 설정하고, StringBuilder에 넣어준뒤 마지막에 출력한다.

##시간 복잡도
-O(n)

##느낀점
-입력을 Scanner로 하고 sout을 사용하면 시간 초과가 된다.<br>
-BufferReader와 StringBuilder를 사용하고 출력을 반복문이 끝난뒤 한번에 출력하면 시간을 효율적으로 개선할 수 있다.<br>
-while문을 돌릴때 n-->0 을 사용하여 코드를 단축시킬 수 있다.<br>
