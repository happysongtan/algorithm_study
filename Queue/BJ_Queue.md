##풀이 문제
- [주소](https://www.acmicpc.net/problem/10845)

##풀이 아이디어
-Queue를 선언하고 명령에 맞게 if를 설정해준다.
-push X: 정수 X 입력
-pop: 앞에 있는 정수 빼기
-size: 사이즈 출력
-empty: 비어있으면 1, 아니면 0
-front: 앞에 있는 정수 출력
-back: 뒤에 있는 정수 출력(back 변수 사용)

##시간 복잡도
-O(N)

##느낀점
-if를 사용하여 조건을 컨트롤 하였는데 if가 너무 많아서 가독성이 떨어졌다.
-명령어 기준으로 분기를 하는 방향으로 로직을 변경할 필요있다.
