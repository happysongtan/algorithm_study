#풀이 문제
- [주소](https://school.programmers.co.kr/learn/courses/30/lessons/42839)

##풀이 아이디어
-문자를 배열로 변환하고, 배열의 크기만큼의 boolean을 선언하여 방문여부를 확인함.
-중복을 피하기 위해 Hashset을 사용하고, 재귀함수를 check_Prime으로 소수인지 확인하는 함수를 isPrime으로 선언한다.
-값을 isPrime으로 확인하여 추가하고 for문으로 checkPrime으로 방문여부를 확인 후 값을 도출한다.

##시간 복잡도
-O(n! x √n)

##느낀점
-재귀를 활용한 완전탐색 문제였지만, 문자열 조합과 소수 판별을 함께 처리하는 과정이 쉽지 않았다.
-소수 판별 시 루트 값을 기준으로 반복문을 줄이는 방식(i * i <= num)이 효율적이라는 것을 다시 확인했다.
-방문 배열을 사용한 후, 재귀 호출이 끝나면 다시 false로 되돌리는 백트래킹 과정이 중요하다는 것을 느꼈다.
